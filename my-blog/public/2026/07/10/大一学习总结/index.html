大一学习总结

Created2026-07-10|Updated2026-07-10|生活随笔
|Post Views:

大一上

Java

高三暑假收到软件工程的录取通知书，开始学习Java基础，了解了Java基本语法。但由于实验室招新考试没过（要求有算法基础），便暂停了Java的学习。

C语言

10月份开始转学C语言，断断续续到12月学完C语言。

数据结构与算法

学完C语言后开始学习数据结构与算法，能够手写顺序表、链表、队列、栈、二叉树、图等常见数据结构。

Python

寒假期间开始学习Python，掌握Python基础语法。使用 Streamlit 和 OpenAI 包跟着黑马程序员写了一个「AI虚拟伴侣」项目，实现了对话存储、流式输出等功能。

📸 项目截图1 - 对话界面 | 项目截图2 | 项目截图3

C++

完成Python项目后开始学习 **C++**，掌握了 STL库 的使用方法，并开始学习一些算法。

由于练习不够、操之过急，4月份蓝桥杯只拿到了省二等奖。📄 查看获奖证书

大一下

JavaSE & MySQL

5月份重新开始学习 JavaSE，月底学完。之后开始学习 MySQL，掌握了数据库的增删改查（CRUD）语句。

SSM框架

MySQL学完后，开始学习 SSM（Spring + SpringMVC + MyBatis） 框架，一直持续到现在。
